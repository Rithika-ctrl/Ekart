Role Permissions Matrix

Overview of access permissions for each role. Use this as a reference when assigning roles.

ADMIN
Full System Access
  • Manage all users & vendors
  • Approve/reject products
  • Content management (banners)
  • Process refunds & returns
  • RBAC / Security settings
  • View platform analytics
Protected Routes
/admin/* /approve-products /admin/refunds /admin/security
ORDER_MANAGER
Order Operations
  • View all orders
  • Update order status
  • Track shipments
  • Handle cancellations
  • No admin panel access
  • Cannot manage users
Protected Routes
/orders/* /track-order/* /cancel-order/*
CUSTOMER
Standard User
  • Browse & search products
  • Add to cart & checkout
  • View own order history
  • Manage addresses
  • No admin access
  • No order management
Accessible Routes
/customer/* /cart /checkout
